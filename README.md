## Node Server 

### Installation

*sudo apt-get install node nodejs-legacy npm*

*npm i express --build-from-source*

*npm i sqlite3 --build-from-source*

*npm i body-parser --build-from-source*  

*npm i ejs --build-from-source*


### Running

*node server.js*

Visit localhost:3000
